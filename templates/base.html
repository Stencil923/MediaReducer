Welcome to MediaReducer
Automatic movie library cleanup for Plex and Jellyfin
This app deletes movie files — there is no recycle bin. Stay on Monitor Only and use Simulate until the output matches exactly what you expect before you delete anything for real.
Do not expose this app to the internet. There is no login — anyone who can reach the web UI can delete your media. Keep port 7474 on your LAN or behind a VPN.
Quick start
  1. Open Configuration; leave Scheduler Mode on Monitor Only.
  2. Connections: pick your servers, use Auto Detect, then Check for Errors.
  3. Movie Library Paths: add the folders MediaReducer may manage and protect any collections.
  4. Set Space Thresholds, then tune scoring on the Filtering & Scoring tab.
  5. Run Simulate from the Dashboard and read the log. Turn on Automatic Cleanup only when it deletes exactly what you'd delete yourself.
About

MediaReducer scores every movie from watch history and IMDb rating, and frees space by deleting the lowest-value files first when your storage thresholds are crossed. Protected collections, filtered movies, and anything it can't safely identify are always skipped. Reopen this guide any time with the ? button in the header; full documentation is in the README.

IMDb ratings data required

IMDb ratings are required for scoring — the copy on disk was missing or older than your IMDb refresh interval, and the automatic download failed. Add the file manually:

  1. Download title.ratings.tsv.gz from IMDb's datasets page.
  2. Drop that file into your config folder as:
    /config/title.ratings.tsv.gz
  3. Run again. MediaReducer unpacks it automatically. You can also place the extracted /config/title.ratings.tsv.
Runs keep the file refreshed after it exists. The next Simulate also annotates the Filtering & Scoring library table with ratings.
